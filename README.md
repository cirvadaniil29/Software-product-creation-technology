# Файли для дисципліни "Технології створення програмних продуктів"
Репозиторій створений студентом 2 курсу групи ІН-42/1 спеціальності «Комп'ютерні науки» Сумського державного університету, факультету ЕлІТ, Чирвою Даніілом для дисципліни «Технологія створення програмних продуктів» з метою виконання завдань.
